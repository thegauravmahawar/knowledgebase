## Basics

| Command | Function |
| --- | --- |
| vim <filename> | Open <filename> in vim |
| :q | Quit vim |
| :w | Save current file |
| :wq | Save file and quit vim |
| ZZ | Save file and quit vim |
| :q! | Quit vim without saving file |
